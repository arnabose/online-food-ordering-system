# Online Food Ordering System

A full-stack web application that enables users to browse food items, place orders, manage their cart, and track orders through a user-friendly interface. The project also includes an admin panel for managing categories, menu items, and customer orders.

Live Demo

Live Website: https://foodians.epizy.com/

## Overview

The Online Food Ordering System is designed to simplify the food ordering process by providing customers with an easy-to-use platform and administrators with efficient management tools. It demonstrates full-stack web development concepts including authentication, database integration, session management, and CRUD operations.

## Installation

1. Clone the repository

```bash
git clone https://github.com/arnabose/online-food-ordering-system.git
```

2. Move the project to your XAMPP `htdocs` folder.

3. Start **Apache** and **MySQL** from the XAMPP Control Panel.

4. Open **phpMyAdmin**.

5. Create a database named:

```text
foodians
```

6. Import the SQL file located in:

```text
database/foodians.sql
```

7. Open your browser and visit:

```text
http://localhost/Online-Food-Ordering-System
```
